# SMS Consent

Andrew Mack Uptime Alerts is a private personal infrastructure monitoring service. It is not open to the public.

Before a phone number is added, the account owner asks the recipient:

> Would you like to receive automated SMS notifications from Andrew Mack Uptime Alerts for personal infrastructure monitoring? These messages may include server outages, service failures, home internet connectivity issues, CCTV connectivity status, and recovery notifications. Message frequency varies based on monitored events. Message and data rates may apply. Reply HELP for assistance or STOP at any time to unsubscribe.

A phone number is added only after the recipient explicitly agrees to receive these messages.

Currently, the only authorized recipients are the account owner and one family member.
