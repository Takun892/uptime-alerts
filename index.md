# Personal Infrastructure Monitoring

This website provides information regarding the SMS notification service used for personal infrastructure monitoring.

Automated notifications are generated by Uptime Kuma for:

- Server availability
- Service outages
- Home Internet connectivity
- CCTV connectivity monitoring

This service is intended only for authorized recipients who have provided consent to receive these notifications.

No marketing, advertising, or promotional messages are sent.
